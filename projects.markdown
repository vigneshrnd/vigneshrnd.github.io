<h4 style="text-align: center;"> Product Surface Area </h4>
 **Systemwide Intelligence** - Systemwide Visual Generation, Visual Intelligence (Lookup, Search), Code Scanning.  <br/>
 **Developer Experience** - 3rd party APIs in Vision.fwk, 3rd party Safety APIs, CreateML / CoreML for custom training.  <br/>
 **Communication Apps** - Live Stickers, Communication Safety, Genmoji, Generative Messages Background.  <br/>
 **Apple Camera Experiences** - Photographic Styles & Semantic Segmentation, Portrait Mode, QR Code Scanning, Live Text. <br/>
 **Apple Photos** - Apple Intelligence Memories, Photos Curation, Photos Search, Personalization. <br/>

<h4 style="text-align: center;"> Core Technologies </h4>
**Visual Generation**: Spearheaded on-device image generation / diffusion, Apple Diffusion Model (ADM) pre-training / post-training and optimization. <br/>
**Infrastructure** - Training workflows for distributed training of diffusion  embedding models, Apple Neural Engine optimized deployment of transformer models. <br/>
**Safety** - Training and deployment of systemwide guardrails that power Apple  Intelligence. Alignment / model mitigation post training. <br/>
**Multimodal Embeddings** - The primary embedding model that is used across Apple operating systems. Also resulted in image/video captioning that allows for efficient descriptions of visual content in performance constrained settings. <br/>
**Child Safety** - Led the development of communication safety models for images and video. <br/>
**Systemwide Visual Perception** - Fundamental image tagging, image saliency estimation, aesthetics, embeddings, custom classifiers are extensively used for visual analysis across the system. <br/>
